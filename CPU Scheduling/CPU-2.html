About CPU Scheduling Algorithms

CPU scheduling is the process of determining which process in the ready queue will be allocated the CPU for execution. The primary objective is to maximize CPU utilization and throughput while minimizing turnaround time, waiting time, and response time.

Algorithms Implemented:

  • Shortest Job First (SJF): A non-preemptive scheduling algorithm that selects the process with the smallest execution time first.
  • Priority Scheduling: Processes are scheduled based on priority. Lower numbers typically indicate higher priority.
  • Shortest Remaining Time First (SRJF): Preemptive version of SJF where the process with the smallest remaining time is selected for execution.
