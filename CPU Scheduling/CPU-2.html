About CPU Scheduling Algorithms

CPU scheduling determines which process in the ready queue is allocated the CPU. The primary objective is to maximize CPU utilization and throughput while minimizing turnaround time, waiting time, and response time.

Algorithms Implemented

  • Shortest Job First (SJF): A non-preemptive algorithm that selects the process with the smallest execution time first.
  • Priority Scheduling: Processes are scheduled based on priority. Lower numbers indicate higher priority.
  • Shortest Remaining Time First (SRJF): Preemptive version of SJF — the process with the smallest remaining time is always selected.
