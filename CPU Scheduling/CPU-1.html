First Come First Serve

Definition

FCFS executes processes in arrival order, running each to completion.

Advantages

  • Simple to implement
  • No starvation

Disadvantages

  • Convoy effect
  • Poor average waiting time

Round Robin

Definition

RR assigns each process a fixed time quantum; if unfinished, it rejoins the ready queue.

Advantages

  • Fair time-sharing
  • Good response time

Disadvantages

  • High context-switch overhead
  • Quantum choice affects performance
