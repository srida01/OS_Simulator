Explore fixed and variable memory partitioning strategies. Observe how the OS allocates memory to processes and understand internal vs. external fragmentation in real time.

Select Simulation Mode
Mode 01 · Fixed Partitioning

MFT

Multiprogramming with a Fixed number of Tasks. Memory is divided into equal-sized partitions before execution. Each process occupies one partition regardless of its actual size — leading to predictable allocation but potential internal fragmentation.

▶ Simulate MFT
Mode 02 · Variable Partitioning

MVT

Multiprogramming with a Variable number of Tasks. Partitions are created dynamically based on each process's exact memory requirement. Maximizes memory utilization but can cause external fragmentation as free blocks become scattered over time.

▶ Simulate MVT
